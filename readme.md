# ToDo Manager - Менеджер задач

Простая консольная программа для управления задачами, написанная на Python.

## Функционал

Добавление задач - создание новых задач с автоматической нумерацией

Удаление задач - удаление по ID

Отметка выполнения - пометка задач как выполненных

Возврат в работу - снятие отметки о выполнении

Просмотр задач - красивый вывод всех задач с статусами

Очистка выполненных - массовое удаление выполненных задач

Автосохранение - автоматическое сохранение в JSON файл

## Особенности

Автоматическая нумерация - ID задач генерируются автоматически

Временные метки - каждая задача содержит дату и время создания

Подтверждение действий - запрос подтверждения для опасных операций

Обработка ошибок - корректная обработка неверного ввода

Кодировка UTF-8 - поддержка русского языка
